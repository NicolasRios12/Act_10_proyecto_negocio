# 🎯 Agente Global .agents - Proyecto Dog Club

Bienvenido a la documentación del agente global para **Dog Club**. Este agente orquesta el desarrollo de la aplicación de gestión de usuarios y cuidadores.

## Estructura del Agente

- `skills/`: Definiciones de arquitectura, diseño y flujo.
- `scripts/`: Automatización de tareas de desarrollo.
- `resources/`: Documentación de soporte técnico.

## Resumen de Funcionalidad

Esta base está lista para:

1. **Administración Centralizada:** El Admin gestiona el flujo de datos.
2. **Gestión de Entidades:** CRUD completo para `Cuidadores` y `Usuarios`.
3. **Persistencia de Datos:** Firestore configurado para un entorno seguro de negocio.
