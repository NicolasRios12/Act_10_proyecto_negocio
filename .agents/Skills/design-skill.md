# 🎨 Skill de Diseño UI

Guías de diseño para **Dog Club**.

## Paleta de Colores

- **Fondo**: `#F0F4F8`
- **AppBar / Iconos**: `#1976D2`
- **Botones**: `#0D47A1`
- **Acentos / Footer**: `#BBDEFB`

## Tipografía

- **Principal**: Roboto o Inter.
- **Jerarquía**: H1 (Negrita), H2 (Semibold), Cuerpo (Regular).

## Componentes UI

- `custom_appbar.dart`: AppBar azul corporativo.
- `custom_buttons.dart`: Botones primarios azul oscuro.
- `list_cards.dart`: Cartas con fondo acento (`#BBDEFB`) para listar registros.
